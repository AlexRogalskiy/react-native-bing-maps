# react-native-bing-maps

React native wrapper for Bing Maps

## Installation

```sh
npm install react-native-bing-maps
```

## Usage

```js
import BingMaps from "react-native-bing-maps";

// ...

const result = await BingMaps.multiply(3, 7);
```
